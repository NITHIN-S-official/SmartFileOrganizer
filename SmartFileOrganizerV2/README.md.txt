# Smart File Organizer

A Python desktop app that organizes files by type. Built using Tkinter.

## Categories
- Word Documents
- PDFs
- Presentations
- Spreadsheets
- Text Files
- eBooks
- Images, Music, Videos, Archives

## How to Use
1. Run `organizer.exe` inside the `dist/` folder.
2. Click “Select Folder”
3. The app will sort all files automatically.
